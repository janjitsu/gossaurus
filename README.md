# GoSSaurus - Yet another Kindle dictionary Generator

### generate a rich dictionary for your kindle

### dependencies: kindlegen_linux

## Usage
```bash
 gossaurus build 
    --from=sr \
    --to=en \
    --title="Serbian to English Dictionary" \
    --author="Milos" \
    --cover="cover.png" \
    --entries=.resources.csv
```

## Install and Config
```aiignore
make install
```

## Building from source
```aiignore
make build
```
